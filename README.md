# Function Machine Fun

Tiny browser lesson for teaching functions to a 1st grader with an input-output machine metaphor.

## Open It

Open [`index.html`](/Users/pankaj/codex-hacks/index.html) in a browser.

If you want to serve it locally instead, run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Files

- [`index.html`](/Users/pankaj/codex-hacks/index.html): page shell
- [`styles.css`](/Users/pankaj/codex-hacks/styles.css): visuals and responsive layout
- [`script.js`](/Users/pankaj/codex-hacks/script.js): reusable lesson scenes and interactions
